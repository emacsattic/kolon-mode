kolon-mode
==========

kolon-mode is a html-mode derived major mode for Text::Xslate's Kolon syntax.

Features
--------
kolon-mode will stay (mostly) out of your way with manual tabbing.

Kolon docs on CPAN are accessible by default with C-c C-k
